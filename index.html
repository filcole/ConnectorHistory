Connector Status

This micro-site hosts an unofficial consolidated list of connectors and actions for Microsoft Power Automate, Microsoft Power Apps, and Azure Logic Apps in a computer-friendly format.

For full documentation about each Power Platform Connector refer to the Microsoft's official connector reference documentation or Connector Review by CRM Heidi.

  • All.json - a list of all connectors and actions
  • Deprecated.json - a list of all connectors and actions that have been deprecated
  • NonDeprecated.json - a list of all connectors and actions that have not been deprecated
  • connectors/<connectorname>.json - a folder containing a file for each connector, e.g. connectors/excelonlinebusiness.json

Refer to the accompanying ConnectorHistory github repo for history of the connectors since 8th August 2021.

This is a community project and maintained with on a best effort basis. Always refer to the official documentation if in doubt. All brand names and copyrights are owned by their respective owners and are used here only in reference to power platform connectors.
